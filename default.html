However, methodological approaches that consider the use of biotic variables during the prediction process are still largely lacking. Most fishing forecasts use weather data coupled with assumptions about how likely a fish is to appear under given conditions. Fishing success is defined as the expected weight of the fish caught. Previous reports have mainly focused on commercial fishing or limited the research to examining the impact of a single variable. Machine Learning models are used to predict the weight of the fish. With this overview, this project provide following contribution. Firstly, the Fish weight feature dataset from UCI machine repository is subjected with the data processing and cleansing. Secondly, the raw data set is applied to find the regression relationship between all the features with fish weight is done with visualization. Thirdly, the raw dataset and the feature scaled dataset is applied to all the Linear Regression models and the performance is analyzed. Fourth, the raw dataset and the feature scaled dataset is applied to all the Ensembling Regression models and the performance is analyzed through intercept, MAE, MSE, R2Score, and EVS.
